正则

用正则给数字添加千分符

举例：将'10000000000'形式的字符串，以每3位进行分隔展示 '10.000.000.000'

8.如下一个字符串"54389"，要求将字符串中的阿拉伯数字替换成我们的中文大写数字"伍肆叁捌玖"，请使用正则的方式进行处理

let str = '54389'

const array = ['零', '壹', '贰', '叁', '肆', '伍', '陆', '柒', '捌', '玖']

str = str.replace(/\d/g, match => {
  return array[match]
})

console.log(str);
